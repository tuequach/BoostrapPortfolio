Portfolio and Projects

Terminal Force

A networked fighting game where player can control the gravity of objects scattered in the level to fight their opponents. Terminal Force was developed on Unity 2D/3D game engine. We worked in a team of 6 and mainly used Slack to communicate, Trello and Github.


Condestruction

Twin stick shooter where players compete to destroy enemies to create platforms to reach the end level. Condestruction was developed on Unity 3D/2D game engine and was submitted to Texas A&M University Chillennium Game Jam Competition. Participants of four was given 48 hours to develope a game from stratch including planning, designing and developing the game based on a given theme of "Transformation".


City Rank

Ranking livable cities based upon user preferences of 3 dimension (civic happiness, political affiliation, cost of living). Front-end Design: focused mainly on UI design using Bootstrap Material Design and what the user sees in the browser. JavaScript, HTML5, CSS3, jQuery, Bootstrap Material Design, Ajax, Firebase.


Green Leaves

Green Leaves is a business-to-business (B2B) full-stack web application enabling customers to place orders with an Asian vegetable wholesaler for pickup or delivery. Worked in a team of 5. Backend Tech-Stack: ReactJS (creating the model data into Sequelize, translation of Vietnamese and Chinese, ExpressJS & API routing, and functional components using React Hooks). JavaScript, Ract, Sequelize ORM, MySQL, RESTful API, NodeJS, ExpressJS Web Server, Material-UI, Auth0, Jest, CircleCI, HTML5, CSS3.
